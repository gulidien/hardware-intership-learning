# 模拟电路学习笔记

## 第一章：半导体器件基础

### 1.1 半导体基本知识

- **半导体**：导电能力介于导体和绝缘体之间的材料，如硅（Si）、锗（Ge）
- **本征半导体**：纯净的半导体，靠热激发产生少量载流子（电子-空穴对）
- **杂质半导体**：
  - **N型**：掺入五价元素（磷、砷），多子为电子
  - **P型**：掺入三价元素（硼、铟），多子为空穴

### 1.2 PN结

- **形成**：P型和N型半导体接触，交界处形成空间电荷区（耗尽层）
- **单向导电性**：
  - 正向偏置（P接高电位）：导通，正向电流大
  - 反向偏置（N接高电位）：截止，反向电流极小

### 1.3 二极管

**伏安特性：**

```
         I（mA）
         |
         |      /
         |     /
         |    /
---------+---/--------→ V（V）
         | /
         |/ ← 反向击穿
```

- **正向导通电压**：硅管约 0.6~0.7V，锗管约 0.2~0.3V
- **主要参数**：最大整流电流 I_F、最高反向工作电压 V_RM

**常用类型：**

| 类型 | 用途 |
|------|------|
| 整流二极管 | AC→DC 整流 |
| 稳压二极管（齐纳管）| 电压稳定 |
| 发光二极管（LED）| 指示/照明 |
| 肖特基二极管 | 高频、低压降 |

### 1.4 三极管（BJT）

- **结构**：NPN 或 PNP，三个区：发射区、基区、集电区
- **三种工作状态**：

| 状态 | 发射结 | 集电结 | 特点 |
|------|--------|--------|------|
| 放大 | 正偏 | 反偏 | $I_C = \beta I_B$ |
| 饱和 | 正偏 | 正偏 | V_CE 很小（≈0.3V）|
| 截止 | 反偏/零偏 | 反偏 | I_C ≈ 0 |

- **电流关系**：I_E = I_B + I_C，I_C = β·I_B（放大区）
