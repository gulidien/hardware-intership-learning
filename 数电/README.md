# 数字电路学习笔记

本目录记录数字电路（数电）的学习过程，包括逻辑代数、组合逻辑、时序逻辑等内容。

## 📂 目录规划

```
数电/
├── 逻辑代数基础/   # 布尔代数、逻辑门
├── 组合逻辑电路/   # 编码器、译码器、多路选择器
├── 时序逻辑电路/   # 触发器、寄存器、计数器
├── 可编程逻辑/     # FPGA/CPLD 入门
├── HDL语言/        # Verilog 基础
└── 实验记录/       # 仿真与实验数据
```

## 📝 学习资源

- 《数字电子技术基础》（阎石）
- Logisim 逻辑仿真软件
- Quartus II / Vivado
