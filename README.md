# hardware-intership-learning

> 硬件方向实习期间的学习记录仓库，涵盖 C 语言编程、模拟电路（模电）、数字电路与微处理器三大模块。

---

## 📂 目录结构

```
hardware-intership-learning/
├── c语言/              # C 语言基础与练习代码
├── 模电/               # 模拟电路学习笔记
├── 数电与微处理器/      # 数字电路与微处理器学习笔记
└── 学习笔记.md         # 综合学习记录
```

---

## 🎯 学习目标

- 掌握 C 语言基础语法，能编写常见算法与数据结构程序
- 理解模拟电路基本原理（放大器、滤波器、运放等）
- 掌握数字逻辑电路设计与分析，了解微处理器工作原理
- 为后续学习 FPGA 设计、单片机开发打下基础
