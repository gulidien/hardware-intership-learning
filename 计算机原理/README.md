# 计算机组成原理学习笔记

本目录记录计算机组成原理的学习过程，包括运算器、存储器、控制器、指令系统等内容。

## 📂 目录规划

```
计算机原理/
├── 数据表示/       # 进制转换、补码、浮点数
├── 运算器/         # ALU设计、加减乘除运算
├── 存储器/         # Cache、主存、虚拟存储器
├── 指令系统/       # 指令格式、寻址方式
├── CPU设计/        # 数据通路、控制单元
└── 总线与IO/       # 总线标准、IO接口
```
