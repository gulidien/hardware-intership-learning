# 求职经历

本目录记录求职过程中的面试准备、投递记录与经验总结。

## 📂 目录规划

```
求职经历/
├── 简历/           # 简历版本与优化记录（不含个人隐私信息）
├── 面试准备/       # 常见面试题整理与解答
├── 投递记录/       # 公司投递情况跟踪
└── 面试复盘/       # 面经记录与经验总结
```

## 📝 投递记录模板

| 日期 | 公司 | 岗位 | 状态 | 备注 |
|------|------|------|------|------|
| YYYY-MM-DD | 公司名称 | 岗位名称 | 投递中/笔试/面试/Offer/拒绝 | 备注信息 |
